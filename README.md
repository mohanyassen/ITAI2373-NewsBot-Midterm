# ITAI2373-NewsBot-Midterm

## Project Overview
The **NewsBot Intelligence System** is an end-to-end NLP project designed to automatically process, analyze, and categorize news articles. This system demonstrates mastery of multiple NLP techniques, including text preprocessing, TF-IDF feature extraction, POS tagging, syntax parsing, sentiment analysis, multi-class classification, and named entity recognition (NER).

The NewsBot system provides insights for media monitoring, business intelligence, and general news trend analysis. It is capable of:

- Preprocessing and cleaning news text  
- Classifying articles into categories: Politics, Sports, Technology, Business, Entertainment, Health  
- Extracting entities such as people, organizations, locations, dates, and monetary values  
- Analyzing sentiment and emotional tone  
- Identifying patterns using TF-IDF, POS tagging, and syntax parsing  

---

## Dataset
The dataset used in this project is the **BBC News Classification Dataset** from Kaggle:

- [BBC News Dataset](https://www.kaggle.com/competitions/learn-ai-bbc/data)  
- Categories: Business, Entertainment, Politics, Sports, Technology  
- Number of articles: ~2,000  
- Language: English  

The dataset was cleaned, preprocessed, and sampled to ensure consistency and completeness.
